# 全自动车牌识别功能说明

## 🚀 功能特点
- **零操作**: 打开摄像头即自动开始识别，无需任何点击
- **连续扫描**: 每2秒自动识别一次，持续监控
- **成功提示**: 识别成功时显示车牌号码，其他时候静默运行
- **实时显示**: 识别结果自动更新到页面上

## 📱 使用方法

### 超简单操作
1. **点击"摄像头识别"** - 摄像头启动后1秒自动开始识别
2. **对准车牌** - 将车牌放在绿色扫描框内
3. **等待识别** - 系统每2秒自动拍照识别
4. **查看结果** - 识别成功会弹出提示，结果显示在页面上

### 手动识别（可选）
- 点击 **📷 手动拍照** 进行单次精确识别
- 适合需要精确控制拍照时机的场景

## 🎯 界面说明

### 状态指示器（右上角）
- **🟢 绿色脉冲点**: 表示自动识别正在运行
- **识别次数**: 显示已执行的识别次数（如：自动识别: 5次）

### 扫描框
- **绿色矩形框**: 车牌识别区域
- **提示文字**: "自动识别中，请保持稳定"

### 控制按钮（底部）
- **📷 手动拍照**: 蓝色圆形按钮，单次识别
- **❌ 关闭摄像头**: 红色圆形按钮，退出识别

## ⭐ 用户体验

### 简洁流畅
- **无弹窗干扰**: 除了识别成功提示，没有其他弹窗
- **自动运行**: 无需手动开启自动识别功能
- **连续监控**: 识别失败时静默继续，不打断流程

### 智能反馈
- **成功提示框**: 识别成功时弹出详细提示框，显示车牌号、颜色、置信度
- **用户选择**: 提示框可选择"继续识别"或"停止"
- **结果更新**: 最新识别结果自动显示在页面上
- **状态清晰**: 通过指示器了解识别进度

## 🔧 技术参数

| 参数 | 值 | 说明 |
|------|----|----- |
| 识别间隔 | 2秒 | 每次自动识别的间隔时间 |
| 启动延迟 | 1秒 | 打开摄像头后开始识别的延迟 |
| 防抖间隔 | 2秒 | 防止过于频繁识别的最小间隔 |
| 图片质量 | high | 拍照时使用高质量模式 |

## 🎯 适用场景

### 理想场景
- **停车场入口**: 无人值守自动识别进出车辆
- **车辆登记**: 批量录入车牌信息，无需重复操作
- **监控系统**: 连续监控特定区域的车辆
- **移动设备**: 手持设备进行车牌扫描登记

### 使用建议
- **保持稳定**: 设备尽量保持稳定，避免剧烈晃动
- **光线充足**: 确保车牌区域有足够的光线
- **对准框内**: 车牌尽量完整地放在绿色扫描框内
- **网络稳定**: 确保网络连接正常以调用识别API

## 📋 操作指南

### 快速上手
```
1. 打开应用 → 点击"摄像头识别"
2. 摄像头启动 → 自动开始识别（1秒后）
3. 对准车牌 → 系统每2秒自动识别
4. 识别成功 → 弹出详细提示框显示车牌信息

```

### 结果查看
- **实时结果**: 页面上"最新识别结果"区域
- **成功提示框**: 详细提示框显示车牌号码、颜色、置信度
- **用户交互**: 可选择继续识别或停止扫描
- **详细信息**: 包含车牌号、颜色、置信度、识别时间

这种全自动的设计让车牌识别变得极其简单，用户只需要对准车牌，系统就会自动完成所有识别工作！
